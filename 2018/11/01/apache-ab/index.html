Linux安装ab压力测试命令

ab -c100 -n1000 http://localhost:4000/

-n requests // 在测试会话中执行的请求个数， 默认时， 仅执行一个请求

-c concurrency // 一次产生的请求个数， 默认为一次一个；

上面的执行命令，表示为：

发送100个请求， 每次行发，1000个请求个数； 访问 后台对应的url
